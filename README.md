# Kerberos + LDAP Authentication Infrastructure

This project demonstrates how to replace Microsoft Active Directory using open source technologies.

## Technologies
- OpenLDAP
- Kerberos
- SSSD
- Linux

## Architecture
LDAP is used as the directory service while Kerberos provides authentication.

## Features
- Centralized authentication
- User directory management
- Single Sign-On (SSO)
